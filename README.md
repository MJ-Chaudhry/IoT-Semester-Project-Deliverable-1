## Background
This repository outlines the research and sample schematic created for the IoT unit semester project deliverable 1 at Strathmore University.

The first section outlines the conditions for growing tulip flowers, such as temperature, humidity, soil type, and so on. The second section outlines the suitable hardware  we will use to create an embedded system for monitoring the environmental measures. The third part links the data sheets for some of the parts listed, and finally, the last part details some template diagram for circuits using a DHT22 and MQ-5 sensor with the ESP32.

## Environmental Factors That Affect Tulips
Below is a list of each of the optimal environmental measures:

| Environmental Property      | Value |
| --------------------------- | ----- |
| Temperature                 |  4°C – 16°C (day), 5°C – 10°C (night)  |
| Relative humidity           |  60% – 80%  |
| Soil type                   |  Well-draining loamy or sandy loam  |
| Soil Moisture Content       |  50% – 70% of field capacity  |
| Soil pH                     |  6.0 – 7.0 (ideal: 6.5)  |
| Number of hours in sunlight |  6 – 8 hours per day  |
